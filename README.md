# gtfs_rt_pb_decoder

- Display GTFS_RT .pb files as human readable text
- Save .pb files as plain.txt files

## Using the application

- Install requirements.txt file.
- Run app.py file.

## Test/Experiments

- Install requirements.txt file.
- Run test_gtfs_feed.py file.
